## 🔧 Personalizações Fáceis

### Adicionar Artigos ao Blog

No `index.html`, localize a seção `<section class="blog">` e substitua os cards `coming-soon` por:

```html
<article class="blog-card">
    <div class="blog-image">
        <img src="artigo-foto.jpg" alt="Título do artigo">
    </div>
    <h3>Título do Artigo</h3>
    <p>Resumo breve do artigo...</p>
    <a href="artigo.html" class="blog-link">Ler mais →</a>
</article>
```

### Alterar Textos

Todos os textos estão no `index.html`. Use Ctrl+F para localizar e editar.

### Alterar Cores

No `styles.css`, modifique as variáveis CSS no início:

```css
:root {
    --color-primary: #2D2952;
    --color-accent: #39FF14;
    /* ... */
}
```

### Adicionar Mais Serviços

Na seção `<section class="solutions">`, adicione mais cards:

```html
<div class="solution-card">
    <h3>Novo Serviço</h3>
</div>
```
